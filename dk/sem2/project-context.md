

### Приложение для заполнения дорожной карты.

структура папок и файлов

- /backend - бекенд приложения реализован с использованием java spring boot

- /frontend - frontend приложение на react

- /database.md - описание структуры базы данных

- /description.md - описание технологического стека используемого для реализации фронтенда и бекенда

- /roadmap_table.md - пример всевозможных вариантов шагов, которые должно поддерживать приложение для создания дорожной карты
- /ext/task-be.md - промпты, которые использовались для создания бекенда
- /ext/task-fe.md - промпты, которые использовались для создания фронтенда



Изучи все эти файлы а так же ознакомься с текущей реализацией фронтенда и бекенда. Изначально приложение поддерживает ограниченное количество шагов. 

обрати внимание на файл /backend/src/main/resources/application.properties в котором реализована конфигурация правил для проверки шагов, описание этой логики есть в задаче №8 в файле с промптами для создания бекенда.
